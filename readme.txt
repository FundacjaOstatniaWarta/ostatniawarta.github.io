# Fundacja Ostatnia Warta - Strona Główna (www.ostatniawarta.pl)

Oficjalne repozytorium kodu witryny internetowej Fundacji "Ostatnia Warta" z siedzibą w Gdańsku (KRS: 0001224367).

## O projekcie
Strona została zaprojektowana w czystym HTML5/CSS3. Pełni rolę informacyjną dla:
1. Podopiecznych i ich rodzin (dostęp do dokumentacji medyczno-prawnej).
2. Wolontariuszy (zasady i rekrutacja do Sitting Service).
3. Darczyńców (wsparcie, CSR, konto bankowe Nest Bank, integracja z FaniMani).

## Struktura plików
- `index.html` - Główny plik strony (one-pager).
- `polityka-prywatnosci.html` - Podstrona z obowiązkiem informacyjnym RODO.
- `zbiorki.html` - Podstrona kierująca do zbiórek zewnętrznych (Siepomaga, Zrzutka).
- `/dokumenty/` (Działy 1, 2, 3) - Wnioski, formularze, wzory umów darowizn.
- `robots.txt` - Instrukcje dla robotów sieciowych.
